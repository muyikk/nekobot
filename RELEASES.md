# Release v1.6.4

## 新增
- 新增功能：
  - 支持将下载的漫画以加密 PDF 发送到指定邮箱
  - 支持每个用户单独配置 SMTP 邮箱服务
  - 私聊 1 秒内多条文本消息自动合并后回复，减少刷屏
  - 修复主动聊天与输入状态上报相关报错，提升稳定性
  - 重构大模型调用逻辑，移除 OpenAI SDK，统一使用 requests
