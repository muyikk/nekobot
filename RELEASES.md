# Release v2.0.0

## 新增
- 简化 `/主动聊天` 指令：无需再输入 `1/0`，直接在当前会话中切换主动聊天开关
- 会话粒度控制：私聊与每个群聊各自独立保存主动聊天状态，互不影响
- 机器人生命成长系统：
  - 每个私聊 / 群聊拥有一条独立的「生命」
  - 生命按阶段成长（婴儿期 → 童年期 → 少年期 → 成年期 → 老年期），并在生命周期结束后轮回重生
  - 当前生命阶段会影响聊天风格与说话方式，但不会在回复中刻意暴露年龄设定

---
*发布日期: 2026-01-01*

# Release v1.6.5

## 优化
- 动画表情识别优化：支持 GIF 动图本地多帧截取与多图识别，提升对动态内容的理解能力
- 稳定性提升：修复 AI 回复中包含大括号但非协议格式时可能导致回复为空的问题
- 依赖更新：新增 `Pillow` 库支持图像处理

# Release v1.6.4
